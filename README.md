# svxui

Free and open-source UI component library for svelte app.

--- 

[Documentation](https://svxui.vercel.app/)

## Features

## Installation

```bash
npm install svxui
pnpm install svxui
yarn add svxui
```

## Development

### Install 

```bash
pnpm install
```

### Serve docs site + lib

```bash
pnpm run dev
```

### Serve lib only

```bash
pnpm run dev:lib
```
